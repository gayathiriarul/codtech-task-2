## overview the project ##

Project Overview: Automated Report Generation

Automated Report Generation is a project that uses Python to read data from a CSV file, analyze it, and automatically generate a PDF report.

Input

CSV file (for example, student names and marks)


Process

The CSV file is read using the Pandas library

Calculations such as total, average, highest, and lowest values are performed

A PDF report is generated using the FPDF library


Output

A well-formatted PDF Report
